


## Clone this repository

``` 
rm -rf .config/nvim
git clone git@github.com:keer2345/nvim-keer.git .config/nvim
```

```
curl -fLo ~/.config/nvim/autoload/plug.vim --create-dirs \
 https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

**Alias:**
``` sh
alias vim='nvim'
```
